# UsedCar_Transaction_Site

요약
-------------

JSP 기초 프로젝트
프로젝트 만들면서 프론트엔드쪽에 필요성을 느낌

JSP를 이용하여 회원들이 직접 판매할 차량을 등록하고
다른 사람들과 중고차 거래를 할 수 있는 사이트 구현

웹 서버로는 Apache Tomcat 사용
DB는 MySQL 사용
